How ContextS Works

Simple, powerful, and intelligent documentation enhancement in three steps

1

Search & Resolve

Use resolve_library_id to find and identify the libraries you need from our extensive Context7 database.

2

AI Enhancement

Call get_smart_docs with your context, and our AI models generate comprehensive, practical documentation.

3

Get Results

Receive production-ready documentation with complete code examples, best practices, and integration patterns.

See It In Action

# Search for a library
await resolve_library_id("next.js")
# Returns: Available libraries with IDs and descriptions
# Get AI-enhanced documentation
await get_smart_docs(
library_id="vercel/next.js",
context="building e-commerce site with auth",
extra_libraries=["supabase/supabase", "stripe/stripe-js"],
model="gemini-2.5-flash"
)
# Returns: Complete integration guide with runnable examples
